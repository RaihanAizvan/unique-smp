# Unique SMP - Visual Identity Guide

## Logo Concept

### Design Direction
The Unique SMP logo should embody:
- **Premium Gaming Brand**: Clean, modern, professional
- **Minecraft Aesthetic**: Subtle voxel/block geometry
- **Red Accent**: Deep crimson as primary brand color
- **Bold Typography**: Strong, memorable wordmark

### Logo Variations Needed
1. **Primary Logo**: Full "UNIQUE SMP" wordmark with icon
2. **Icon Only**: Standalone symbol for social media
3. **Horizontal**: For headers and wide spaces
4. **Vertical**: For square placements

### Color Palette

#### Primary Colors
- **Deep Black**: `#0a0a0a` - Main background
- **Charcoal**: `#121212` - Secondary surfaces
- **Crimson Red**: `#dc2626` - Primary accent, CTAs
- **Bright Red**: `#ef4444` - Hover states

#### Secondary Colors
- **Pure White**: `#ffffff` - Primary text
- **Light Gray**: `#d4d4d4` - Body text
- **Medium Gray**: `#737373` - Captions
- **Dark Gray**: `#525252` - Disabled states

#### Accent Colors (Minecraft-inspired)
- **Grass Green**: `#7cb342` - Subtle accents
- **Stone Gray**: `#6b7280` - Neutral elements
- **Diamond Blue**: `#3b82f6` - Rare use, Bedrock
- **Gold**: `#fbbf24` - Highlights

### Typography

#### Primary Font: Inter
- **Use**: All body text, UI elements, navigation
- **Weights**: 300 (Light), 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold), 900 (Black)
- **Source**: Google Fonts

#### Display Font: Minecraft/Pixel-Inspired (Optional)
- **Use**: Headings only, sparingly
- **Style**: Blocky, geometric, subtle pixel feel
- **Recommendation**: Consider custom wordmark instead of full pixel font

### Logo Creation Guidelines

#### Concept 1: Geometric Voxel
- Abstract cube/block arrangement forming "U" or "SMP"
- Clean geometric shapes
- Red gradient with subtle glow
- Modern, minimalist

#### Concept 2: Wordmark + Icon
- Bold sans-serif "UNIQUE SMP"
- Small Minecraft block icon as dot on "i" or separate element
- Clean, professional, gaming-focused

#### Concept 3: Shield Emblem
- Shield shape (protection/community)
- Voxel elements within
- "UNIQUE SMP" text integrated
- Premium, clan-like aesthetic

### File Formats Needed
- **SVG**: Scalable for all sizes
- **PNG**: Transparent background (1024x1024, 512x512, 256x256)
- **Favicon**: 32x32, 16x16

### Usage Rules
- **Clear Space**: Minimum padding equal to height of "U" in logo
- **Minimum Size**: Never smaller than 120px width
- **Background**: Always on dark backgrounds (#0a0a0a or darker)
- **Color Variations**: Full color (primary), white (dark backgrounds), black (light backgrounds if ever used)

## Visual Style Guidelines

### Photography/Imagery
- High-quality Minecraft screenshots
- Cinematic angles and lighting
- Red accent overlays or highlights
- Focus on builds, community, epic moments

### UI Elements
- **Corners**: Slight rounding (8-12px radius) for premium feel
- **Borders**: Thin (1-2px), subtle, often with red accent
- **Shadows**: Soft, dark, with optional red glow
- **Spacing**: Generous, Apple-like breathing room

### Animation Principles
- **Duration**: 300-500ms for most transitions
- **Easing**: Smooth, natural curves (not linear)
- **Purpose**: Always functional, never gratuitous
- **Performance**: 60fps minimum
